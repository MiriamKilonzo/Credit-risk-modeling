# Predicting Loan Default Risk at FinTech Innovations

A machine learning pipeline built to modernize loan approval decisions using 20,000 historical applications.

## Project Summary
- **Approach:** Binary classification (Loan Approved: Yes/No)
- **Final Model:** Tuned Gradient Boosting Classifier
- **F1 Score:** 0.921 | **ROC-AUC:** 0.993 | **Net Business Value:** $3,190,000

## Files
- `financial_loan_risk.ipynb` : Full analysis notebook (CRISP-DM framework)
- `financial_loan_data.csv` : Dataset (20,000 loan applications)

## Libraries
pandas, numpy, scikit-learn, matplotlib, seaborn, scipy

## Framework
CRISP-DM: Business Understanding -> Data Understanding -> Data Preparation -> Modeling -> Evaluation
